What's MerceEdge?

MerceEdge is an IOT edge heterogeneous protocol connection & computing framework that can run on PC, RaspberryPi, and server. It can be used in a variety of applications, such as SmartHome, SmartFactroy, etc. Currently, the protocols currently planned include: mqtt, ZigBee, BLE, REST API.
